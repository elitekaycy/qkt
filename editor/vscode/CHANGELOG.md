# Changelog

## 0.1.0 — Initial release

- Syntax highlighting for `.qkt` files (bundles the project's TextMate grammar).
- Bracket matching and auto-pair for `{}`, `[]`, `()`, `""`, `''`.
- Line comments (`--`) and block comments (`/* ... */`).
- Indent rules for `SYMBOLS` / `RULES` / `DEFAULTS` / `LET` / `FOR EACH` blocks.
- Starter snippets: `strategy`, `sym`, `rule`, `buy`, `buybr`, `pctrisk`, `cross`, `let`, `def`, `foreach`, `flatten`, `notnull`.
- No language server integration yet — see [#84](https://github.com/elitekaycy/qkt/issues/84).
- Not yet published to the VSCode marketplace — install via `.vsix`.
